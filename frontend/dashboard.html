About SkyGuard AI

Advanced machine learning platform for real-time flight risk assessment and anomaly detection. Powered by cutting-edge AI models to ensure aviation safety.

ML Models

  • 🎯 XGBoost - Risk Classification
  • 🧠 Isolation Forest - Anomaly Detection
  • 📊 K-Means - Flight Pattern Clustering
  • ⚡ LSTM Autoencoder - Sequence Anomaly Detection
  • 🛫 LSTM Predictor - Trajectory Forecasting

Data Sources & APIs

  • 🌤️ OpenWeather API - Weather Conditions
  • 🗺️ Leaflet + CARTO Maps
  • 📈 Chart.js - Data Visualization

Technology Stack

  • 🐍 Python + Flask (Backend)
  • 🔬 Scikit-learn + XGBoost + TensorFlow
  • 🎨 Vanilla JavaScript (Frontend)
  • 🗄️ SQLAlchemy + SQLite
  • 🔐 JWT Authentication + RBAC
